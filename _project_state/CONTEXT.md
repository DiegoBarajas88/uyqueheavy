# Contexto del proyecto

UY QUE HEAVY es una marca de barajas conversacionales diseñadas para generar conversaciones profundas, restaurar vínculos y facilitar la expresión emocional.

## Objetivo

Convertir la experiencia física de cartas en una experiencia digital interactiva, mobile-first y premium que permita generar conversaciones reales sin depender del producto físico.

## Audiencia

- Parejas que buscan reconectar
- Amigos que quieren profundizar su relación
- Familias que desean crear momentos significativos
- Usuarios individuales en búsqueda de introspección

## Problemas que resuelve

- No saber cómo empezar conversaciones profundas
- Relaciones en piloto automático
- Falta de conexión emocional
- Dificultad para expresarse
- Miedo a vulnerabilizarse
- Relaciones superficiales
- Desconexión dentro de pareja o familia
- Saturación digital sin conexión real

## Requisitos técnicos

- Next.js App Router
- TypeScript
- Tailwind CSS
- Arquitectura limpia y escalable
- Componentes reutilizables
- Mobile-first obligatorio

## Requerimientos de UX

- Experiencia emocional premium
- Minimalista, elegante, editorial
- Mucho aire y tipografía moderna
- Evitar diseño tipo juego infantil
- UI simple y profunda
