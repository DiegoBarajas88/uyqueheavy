# Handoff Prompt

Continúa el desarrollo de UY QUE HEAVY con el siguiente contexto:

- Proyecto: experiencia digital de cartas conversacionales.
- Stack: Next.js App Router, TypeScript, Tailwind CSS.
- Objetivo: versión móvil premium que facilite conversaciones reales y profundas.
- Requerimientos importantes: no se debe sentir como un juego, debe ser emocional, minimalista y con mucho aire.
- Estructura actual:
  - `/app/page.tsx` — landing emocional.
  - `/app/editions/page.tsx` — selector de ediciones.
  - `/app/experience/[edition]/page.tsx` — experiencia interactiva.
  - `/components/*` — componentes reutilizables.
  - `/data/cards.ts` — datos de cartas estructurados.
- Family Edition: las preguntas se extrajeron desde `Producto/Preguntas Family Edition _ JENGA.xlsx` y se agruparon en categorías de forma responsable.
- Próximo paso ideal: instalar dependencias con Node 16+, ejecutar `npm run dev` y ajustar diseño según branding real.
